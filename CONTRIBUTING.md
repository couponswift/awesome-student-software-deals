To maintain the quality of this list, please ensure your contribution adheres to the following guidelines.

## Verification Rules

We strictly verify every deal to ensure it is valuable for students.

1.  **Student Exclusive**: The deal must be specifically for students (or a generous free tier useful for students).
2.  **Direct Links Only**: Link directly to the official vendor page (e.g., `adobe.com/students`).
    *   ‚ùå No blog posts.
    *   ‚ùå No third-party affiliate links.
    *   ‚ùå No "coupon code" sites.
3.  **Active Offers**: Please check that the deal is currently active.

## How to Contribute

1.  Fork the repository.
2.  Add your link to the appropriate category in `README.md`.
3.  Follow the format: `- [Name](URL) - Description.`
4.  Keep the description short and factual.
5.  Submit a Pull Request!

## Updating Links
If you find a broken link, please submit a PR to fix it or remove the item if the deal no longer exists.
